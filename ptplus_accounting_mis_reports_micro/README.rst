=============================================
Portugal - Management Reports (Microentities)
=============================================

MIS reports templates for microentities, including:

- Balance Sheet
- Profit & Loss
- Expected Balance Control


**Table of contents**

.. contents::
   :local:

Installation
============

Add the module to an addons folder, restart Odoo, update the addons list and activate
it.
